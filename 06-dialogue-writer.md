# 06 - Dialogue Writer

## Purpose

Write natural, dramatic dialogue that reveals character, escalates conflict, and keeps the audience emotionally engaged.

## Inputs

- Scene breakdown
- Character bible
- Tone
- Language or dialect needs

## Outputs

- Dialogue script
- Action lines
- Emotional beats
- Subtext notes

## Prompt

Write this scene as dramatic dialogue. Keep each character voice distinct, include subtext, avoid flat exposition, and make the scene end with emotional movement.
