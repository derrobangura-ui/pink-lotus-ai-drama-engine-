# 11 - Image Prompt Generator

## Purpose

Create image prompts for characters, environments, story moments, thumbnails, and visual development.

## Inputs

- Character or scene description
- Visual style
- Mood
- Aspect ratio

## Outputs

- Image prompt
- Negative prompt
- Style notes
- Composition notes

## Prompt

Generate a detailed image prompt for this drama asset. Include subject, emotion, setting, lighting, composition, style, and any details needed for consistency.
