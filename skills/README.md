# Pink Lotus AI Drama Engine

A production system for turning story ideas into serialized AI drama content, from story analysis through dialogue, visuals, video prompts, thumbnails, social content, and quality control.

## Structure

- `skills/` contains the core production modules.
- `templates/` contains reusable planning documents.
- `workflows/` contains step-by-step production paths.

## Core Pipeline

1. Analyze the story DNA.
2. Build character and series bibles.
3. Plan episodes and scenes.
4. Write emotionally engaging dialogue.
5. Add cliffhangers and retention hooks.
6. Localize for Nigerian audience and cultural context.
7. Generate image, video, thumbnail, and social prompts.
8. Run quality control before production.
