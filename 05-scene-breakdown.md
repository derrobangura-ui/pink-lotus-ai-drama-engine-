# 05 - Scene Breakdown

## Purpose

Break an episode into production-ready scenes with location, action, emotion, and dialogue purpose.

## Inputs

- Episode plan
- Character bible
- Target runtime
- Visual style

## Outputs

- Scene objective
- Location
- Characters present
- Emotional turn
- Key actions
- Dialogue intention

## Prompt

Break this episode into clear scenes. For each scene, identify the goal, conflict, emotional turn, and production notes needed before dialogue writing.
