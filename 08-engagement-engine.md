# 08 - Engagement Engine

## Purpose

Improve retention, comments, shares, and audience curiosity without weakening the story.

## Inputs

- Episode script
- Target platform
- Audience profile
- Desired reaction

## Outputs

- Opening hook
- Comment triggers
- Retention beats
- Shareable moments
- CTA options

## Prompt

Improve this drama episode for engagement. Add stronger hooks, comment-worthy moments, retention beats, and emotional questions while keeping the story believable.
