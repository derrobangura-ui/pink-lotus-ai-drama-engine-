# 02 - Character Bible

## Purpose

Create consistent, emotionally rich characters who can sustain drama across many episodes.

## Inputs

- Story DNA
- Character names or roles
- Relationship map
- Genre and tone

## Outputs

- Character profile
- Motivation
- Secret
- Fear
- Wound
- Voice style
- Relationship conflicts

## Prompt

Build a character bible for this drama. Make each character distinct, emotionally motivated, and useful for conflict, dialogue, and long-term story development.
