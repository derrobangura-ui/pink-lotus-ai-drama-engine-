# 07 - Cliffhanger Engine

## Purpose

Create strong episode endings and scene hooks that make viewers want the next part immediately.

## Inputs

- Episode plan
- Character secrets
- Current conflict
- Platform pacing

## Outputs

- Cliffhanger options
- Reveal options
- Betrayal options
- Emotional hook lines

## Prompt

Generate cliffhanger endings for this episode. Prioritize emotional shock, unanswered questions, reversals, secrets, and consequences.
