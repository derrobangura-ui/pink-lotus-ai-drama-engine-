# Pink Lotus AI Drama Engine

This repository contains the skill files, templates, and workflows for turning any story script into a dialogue-based animated drama production package.

## Main Workflow

1. Analyze the story
2. Build the character bible
3. Create the series bible
4. Plan episodes
5. Break episodes into scenes
6. Write dialogue
7. Localize for Nigerian drama tone
8. Generate animation prompts
9. Generate thumbnails and social content
10. Run quality control
