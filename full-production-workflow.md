# Full Production Workflow

1. Analyze the story.
2. Build the character bible.
3. Build the series bible.
4. Plan the episode.
5. Break down scenes.
6. Write dialogue.
7. Add cliffhangers and engagement hooks.
8. Localize for Nigerian audience needs.
9. Direct animation.
10. Generate image and video prompts.
11. Create thumbnails and social content.
12. Run quality control.
